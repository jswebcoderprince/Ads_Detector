# Ads Detector

A Website Intelligence Platform for Detecting Advertising, Tracking,
Analytics, and Marketing Technologies.

------------------------------------------------------------------------

## Overview

Ads Detector is a website analysis platform designed to identify
advertising technologies, tracking systems, analytics platforms, and
marketing integrations used by websites.

The system analyzes websites using automated scanning techniques and
provides actionable intelligence through evidence-based detection.

------------------------------------------------------------------------

# Features

## Website Technology Detection

-   Advertising platforms
-   Tracking pixels
-   Analytics systems
-   Marketing scripts
-   Third-party integrations

## CSV Processing

-   CSV upload
-   Data validation
-   Duplicate removal
-   Domain normalization
-   Bulk processing

## Website Scanning

The scanner analyzes:

-   HTML information
-   JavaScript resources
-   Network requests
-   Cookies
-   Technology signals

## Evidence-Based Detection

Each result contains:

-   Detected technology
-   Detection evidence
-   Confidence score
